Passo a Passo: Criar Repositório no GitHub

1. Crie uma Conta no GitHub (se ainda não tiver)

  • Acesse GitHub.
  • Cadastre-se com seu e-mail e senha.

2. Acesse o Painel do GitHub

  • Faça login em sua conta.
  • Clique no botão verde "New" para criar um novo repositório.

3. Configure o Repositório

  • Preencha os seguintes detalhes:
    • Repository Name: Nome do repositório (ex.: portfolio-login).
    • Description: (opcional) Adicione uma descrição.
    • Escolha Public (público) ou Private (privado).
    • Marque a opção Add a README file para facilitar o início.
  • Clique em Create Repository.
